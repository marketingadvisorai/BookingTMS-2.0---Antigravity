## Beta Owner Permissions

### âœ… HAS ACCESS TO:
- **Dashboard** - View stats and overview
- **Bookings** - Full CRUD (create, read, update, delete)
- **Activities** - Manage escape room experiences
- **Venues** - Manage up to 3 venues
- **Booking Widgets** - Calendar widgets only
- **Embed Pro** - Create and manage embed configs
- **Customers** - View and manage customer data
- **Waivers** - Important for escape room liability
- **Reports** - View business insights
- **Payments** - View payment history
- **AI Agents** - View status only (no settings)
- **Settings** - Basic configuration

### âŒ NO ACCESS TO:
- System Admin Dashboard
- Organizations Management
- Staff Management
- Campaigns
- Marketing Tools
- Advanced AI Agent Settings
