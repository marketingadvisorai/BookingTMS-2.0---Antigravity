
  # Booking TMS Beta Dev V 0.1

  This repository contains the Booking TMS Beta Dev V 0.1 codebase.

  ## Waiver v0.1.0 — What’s New
  - “Show QR” mode in Scan Waiver with template selection and QR rendering
  - Camera QR scanning via `@zxing/browser` with waiver code verification
  - Upload image QR decoding
  - Manual waiver code verification backed by `localStorage`

  See `CHANGELOG.md` for full release notes.

  ## Running the code

  - Run `npm i` to install the dependencies.
  - Run `npm run dev` to start the development server.
